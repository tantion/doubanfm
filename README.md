# Douban FM Improve

A chrome extension for Douban FM

播放和下载的 kbps 码率，对于普通用户一般都是 64 kbps，对于 pro 用户取决于你账户的设置。

## Features

- 支持点击下载 mp3 和专辑封面
- 支持单曲循环播放
- 支持查看下一首将要播放的歌曲以及切换到下一首
- 支持对分享链接使用私人频道播放
- 支持循环播放专辑
